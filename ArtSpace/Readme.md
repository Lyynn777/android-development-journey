## Art Space App 🎨

### Overview
The Art Space App is an Android application that displays different artworks along with their titles and descriptions.  
Users can navigate between artworks using Next and Previous buttons.

This project helped me understand how to display images and manage UI navigation using Jetpack Compose.

---

### Features
- Displays artwork images
- Shows artwork title and description
- Navigation buttons to switch between artworks
- Simple and structured user interface

---

### Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose

---

### Concepts Learned
Through this project I learned:

- Displaying images using Image composables
- Creating layouts using Column and Row
- Handling button click events
- Updating UI using state variables
