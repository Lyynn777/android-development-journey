## Lemonade App 🍋

### Overview
The Lemonade App is a simple interactive Android application built using Kotlin and Jetpack Compose.  
The app simulates the process of making lemonade by guiding the user through multiple steps such as picking a lemon, squeezing it, drinking lemonade, and restarting the process.

This project helped me understand state management and user interaction in Jetpack Compose.

---

### Features
- Interactive step-by-step lemonade making process
- Image changes based on user actions
- Tap interaction to move between steps
- Simple and clean user interface

---

### Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- Material Design

---

### Concepts Learned
Through this project I learned:

- Managing UI state in Jetpack Compose
- Handling user clicks and interactions
- Updating UI dynamically
- Structuring layouts using Column and Modifier
